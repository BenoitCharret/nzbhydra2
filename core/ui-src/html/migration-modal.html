Migration from NZBHydra 1

You are about to migrate from an installation of NZBHydra 1. This will overwrite your existing settings and database but a backup will be made before.
The migration can take a couple of minutes for huge databases. All enabled newznab indexers will be queried multiple times for a capabilities check.

The old instance of NZBHydra 1 needs to be running and must be at least version 0.2.219. Please enter its base URL below. If it's auth protected please enter the URL like this: http://user:pass@127.0.0.1:5075.
